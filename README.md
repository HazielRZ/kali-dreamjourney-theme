To apply the theme on your Kali Terminal
Para aplicar la configuración completa de manera automatizada, ejecute el siguiente comando en su terminal:

```bash
git clone [https://github.com/hazielrz/kali-dreamjourney-theme.git](https://github.com/hazielrz/kali-dreamjourney-theme.git)
cd kali-dreamjourney-theme
chmod +x install.sh
./install.sh
